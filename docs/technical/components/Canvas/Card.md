# Card Component

## Описание
Компонент карточки-лицензии. Отображает данные партнёра: PV, баланс, актив-заказы, цикл/этап.

## Расположение
`src/components/Canvas/Card.vue`

## Props
| Prop | Тип | Обязательный | Описание |
|------|------|------|------|
| `card` | Object | да | Объект карточки из cardsStore |
| `isSelected` | Boolean | нет (default: false) | Выделена ли карточка |
| `isConnecting` | Boolean | нет (default: false) | Режим рисования соединений |

## Emits
| Событие | Payload | Описание |
|---------|---------|----------|
| `card-click` | `(event, cardId)` | Клик по карточке |
| `start-drag` | `(event, cardId)` | Начало перетаскивания |
| `add-note` | `cardId` | Добавление/открытие заметки |
| `pv-changed` | `cardId` | Изменение PV (при клике на монетку) |
| `open-editor` | `cardId` | Двойной клик — запрос на открытие модала редактирования |

## Интерактивные элементы на карточке
- **Кнопка удаления ×** (`card-close-btn`) — удаление карточки
- **Кнопка заметки** (`card-note-btn`) — открытие окна заметки
- **Монетка** (`coin-icon`) — переключение синий ↔ золотой (меняет PV)
- **Заголовок** — двойной клик для редактирования текста

## Отображаемые значения (только чтение)
- **PV**: левое значение / 330pv
- **Баланс**: L / R (с анимацией при увеличении)
- **Актив-заказы**: L / R (с анимацией при увеличении)
- **Цикл/этап**: numbers

## Редактирование данных
Редактирование PV, баланса и актив-заказов выполняется через **CardEditorModal** — модальное окно, открывающееся по двойному клику на теле карточки.

## Анимации
- `value--animating` — пульсация числа при увеличении значения
- `card--balance-highlight` — подсветка при изменении Active PV
- `card--balance-propagation` — анимация распространения вверх по структуре
- `cardHighlight` — подсветка при навигации от партнёра

## Варианты карточек
- **Обычная** — стандартный размер
- **Большая** (`card--large`) — увеличенные шрифты, аватар партнёра
- **Золотая** (`card--gold`) — золотой стиль, увеличенные шрифты
