# Reset Password (overlay flow)

## Назначение

Экран сброса пароля открывается по ссылке вида `/reset-password?token=...` и отображается как оверлей с формой сброса пароля поверх текущего layout.

## Поведение после успешного сброса

1. Компонент `ResetPasswordForm` выставляет локальный флаг успеха и по кнопке «Готово» эмитит событие `success`.
2. Обработчик в `src/App.vue`:
   - закрывает оверлей сброса пароля,
   - очищает сохранённый reset-токен,
   - делает `router.replace` на главную (`name: 'home'`), с fallback на `/`.

Это предотвращает зависание на `/reset-password` и не оставляет этот маршрут в истории браузера.

## Связанные файлы

- `src/App.vue` — обработчик `handleResetPasswordSuccess`.
- `src/components/ResetPasswordForm.vue` — эмитит `success` по кнопке «Готово».
